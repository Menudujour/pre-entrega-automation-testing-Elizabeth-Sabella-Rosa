# pre-entrega-automation-testing-Elizabeth-Sabella-Rosa

Este repositorio contiene la pre-entrega de pruebas automatizadas para la plataforma **Saucedemo**, desarrollada en **Python** utilizando **Pytest** y **Selenium WebDriver**.

## 📁 Estructura del Proyecto

* `tests/`: Contiene los scripts de prueba principales (ej. `test_saucedemo.py`).
* `utils/`: Funciones de soporte y configuración de helpers.
* `reports/`: Carpeta donde se guardan los reportes de ejecución en formato HTML.
* `requirements.txt`: Archivo con las dependencias del proyecto.
